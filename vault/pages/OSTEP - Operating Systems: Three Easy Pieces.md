## Summary
Free online textbook covering the three core OS concepts: virtualization (processes, address spaces), concurrency (threads, locks, condition variables), and persistence (file systems, devices). Includes homework simulations.

## Key Takeaways
- The OS virtualizes CPU (processes) and memory (address spaces) for each program
- Locks and condition variables are the fundamental concurrency primitives
- File systems trade durability guarantees for performance via journaling
