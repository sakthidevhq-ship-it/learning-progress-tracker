## Summary
Microsoft Research's Suphx achieves superhuman performance in Mahjong on the Tenhou platform. Notable for handling a game with imperfect information, stochastic elements, and 4 players. Uses oracle guiding and global reward prediction.

## Key Takeaways
- Oracle guiding uses perfect-information training then transfers to imperfect-info play
- 4-player imperfect info + stochasticity is significantly harder than 2-player games
- Global reward prediction helps the agent reason about long-term consequences
