## Summary
Build a complete computer system from NAND gates up: logic gates, ALU, CPU, memory, assembler, VM, compiler, and OS. Each layer builds on the previous, demystifying how software meets hardware.

## Key Takeaways
- Every computation reduces to NAND gates — everything else is abstraction
- The fetch-decode-execute cycle is the heartbeat of all CPUs
- A compiler is just a series of syntax-directed translations
